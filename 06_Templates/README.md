# Templates

This folder contains **reusable note templates**.

## Available Templates

- **`daily-note.md`**: For daily journaling in `01_Daily/`
- **`atomic-note.md`**: For evergreen notes in `02_Notes/`
- **`research-note.md`**: For source summaries in `03_Research/`
- **`writing-draft.md`**: For long-form work in `04_Writing/`
- **`review.md`**: For evaluations in `05_Reviews/`

## How to Use Templates

1. **Copy the template** to the appropriate folder
2. **Rename the file** with a descriptive name
3. **Fill in the sections** as needed
4. **Remove or modify** parts that don't apply

Templates are starting points, not requirements.

## Customising Templates

Feel free to:
- Add or remove sections
- Change front-matter fields
- Adapt to your workflow

Save modified templates back here if you want to reuse them.

## AI Assistant Role

An AI can:
- Follow template structure when creating new notes
- Suggest template improvements
- Fill in template sections based on your input

See `.agents/` for skill-specific guidance.
